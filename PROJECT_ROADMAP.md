# Project Roadmap

## 專案目標

建立一套專門針對大盤揭棋的 AI 分析工具。長期目標是實戰輔助分析器，而不是一般象棋 AI。

最終希望支援：

- 分析當前局面最佳步
- 30 秒實戰倒數
- 天天象棋揭棋模式
- 暗子機率推理
- 同步上一手
- 實戰輔助分析流程

## Phase 1：基礎引擎

狀態：進行中，核心已具備。

已完成：

- 棋盤顯示
- 基本走法規則
- 暗子依 `originalType` 走法
- 翻開後依 `realType` 走法
- 合法步生成
- 將軍判定
- 將死判定
- 將帥照面限制
- 撤銷一步
- 基礎局面編輯

仍需補強：

- 更完整的局面編輯，例如新增棋子、指定陣營、載入/儲存局面
- 將死後更新 `GameState.status`
- 更完整的測試局面資料集

## Phase 2：公平分析

尚未開始。

目標：

- Belief State
- 剩餘暗子池
- 各暗子可能身份
- 機率分布
- 已知被吃暗子
- 公平 AI 視角，禁止偷看敵方暗子 `realType`

## Phase 3：實戰模式

尚未開始。

目標：

- 30 秒倒數
- 同步上一手
- 翻子快捷鍵
- Ponder 預分析
- 快取
- 推薦走法卡

## Phase 4：自動化

尚未開始。

目標：

- 天天象棋截圖辨識
- 自動同步
- 行動版優化

目前共識：天天象棋同步功能優先於 OCR，30 秒實戰反應速度優先於極限棋力。
