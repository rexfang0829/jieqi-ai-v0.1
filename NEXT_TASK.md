# Next Task

## 下一輪目標

建議下一輪繼續 Phase 1，做「翻子快捷鍵」的最小版本。

## 為什麼要做

手動抄天天象棋局面時，暗子翻開很常發生。如果每次都要用滑鼠到局面編輯器下拉選單改 `realType` 和 `revealed`，速度會太慢。加入快捷鍵後，手動同步實戰局面會更接近實戰需求。

## 具體要改哪些功能

建議範圍：

- 選取一個棋子後，支援快捷鍵設定該棋子的真實棋種並翻開：
  - `1` = 車
  - `2` = 馬
  - `3` = 象 / 相
  - `4` = 士 / 仕
  - `5` = 炮
  - `6` = 兵 / 卒
- 快捷鍵只作用於目前選取的棋子。
- 快捷鍵執行後：
  - `realType` 更新。
  - `revealed` 變成 `true`。
  - `status` 回到 `playing`。
- 新增測試確認快捷鍵對應的資料轉換邏輯。

## 不要做哪些超出範圍的事

本任務不要做：

- Belief State
- Monte Carlo
- OCR
- Ponder
- 自動截圖辨識
- 雲端同步
- 多局面管理
- 大型重構

## 完成標準

- 選取棋子後可用 1-6 快速翻成指定真實棋種。
- 空格或未選取棋子時快捷鍵不應造成錯誤。
- `npm test` 通過。
- `npm run build` 通過。
- 更新 `CODEX_STATUS.md` 與 `NEXT_TASK.md`。
- 測試與 build 通過後 commit 並 push。
