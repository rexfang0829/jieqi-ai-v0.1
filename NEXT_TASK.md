# Next Task

## 下一輪目標

建議下一輪繼續 Phase 1，做「手動同步上一手」的最小版本。

## 為什麼要做

目前已經可以手動建立局面、設定輪到哪方、儲存 / 載入局面、用快捷鍵翻子，棋盤也已改成交叉點視覺。下一步可以開始靠近天天象棋同步流程：用手動方式點出上一手的起點和終點，讓系統套用這一步。

## 具體要改哪些功能

建議範圍：

- 新增一個「同步上一手」模式開關。
- 在同步模式下：
  - 第一次點棋盤交叉點 = 起點。
  - 第二次點棋盤交叉點 = 終點。
  - 系統嘗試套用這一步。
  - 成功後退出同步模式或清空起終點。
- 同步完成後：
  - 可繼續走棋。
  - 可繼續編輯局面。
  - 可繼續儲存 / 載入。
- 新增測試確認同步套用一步的資料邏輯。

## 不要做哪些超出範圍的事

本任務不要做：

- Belief State
- Monte Carlo
- OCR
- Ponder
- 自動截圖辨識
- 雲端同步
- 多局面管理
- 大型重構

## 完成標準

- 可用手動點選起點 / 終點同步上一手。
- 不使用 OCR 或自動截圖。
- `npm test` 通過。
- `npm run build` 通過。
- 更新 `CODEX_STATUS.md` 與 `NEXT_TASK.md`。
- 測試與 build 通過後 commit 並 push。
