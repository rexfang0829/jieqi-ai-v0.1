# Next Task

## 下一輪目標

建議下一輪仍維持 Phase 1 / Phase 1.5 小範圍工作，優先做「局面手動輸入體驗微調」。

## 為什麼要做

目前已經有局面編輯、儲存載入、手動輪到誰、同步上一手、翻子快捷鍵與長按修正。下一步可以先把實戰手動抄局面的操作變得更穩，不急著進入大型 AI。

## 具體可做功能

- 檢查長按修正選單在手機尺寸上的操作手感。
- 補一個簡短狀態提示，告訴使用者修正成功或因公開棋子超量而失敗。
- 整理目前仍然亂碼的 UI 文案，避免影響操作判讀。
