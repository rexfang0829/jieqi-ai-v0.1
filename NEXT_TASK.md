# Next Task

## 下一輪目標

建議下一輪繼續 Phase 1，做「同步上一手的可視化細節補強」或「局面編輯體驗收斂」的小範圍任務。

## 為什麼要做

目前已經可以手動同步上一手，但實戰輸入時會需要更清楚地知道自己目前在哪個模式、剛剛同步是否成功、以及錯誤時如何重試。這些都是 Phase 1 的互動穩定性，不涉及 AI 推理或自動化。

## 具體要改哪些功能

建議範圍擇一：

1. 同步上一手可視化補強
   - 成功同步後短暫顯示「已同步」。
   - 非法同步時保留錯誤提示直到下一次點起點。
   - 起點高亮更明顯。

2. 局面編輯體驗收斂
   - 新增「清除選取」按鈕。
   - 局面編輯後可顯示簡短提示。
   - 儲存 / 載入後顯示簡短提示。

## 不要做哪些超出範圍的事

本任務不要做：

- Belief State
- Monte Carlo
- OCR
- Ponder
- 自動截圖辨識
- 雲端同步
- 多局面管理
- 大型重構

## 完成標準

- 只改善 Phase 1 UI 互動，不改核心資料模型。
- `npm test` 通過。
- `npm run build` 通過。
- 更新 `CODEX_STATUS.md` 與 `NEXT_TASK.md`。
- 測試與 build 通過後 commit 並 push。
